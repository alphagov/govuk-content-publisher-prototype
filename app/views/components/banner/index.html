Components Banner

Displays a page banner, designed to highlight important information.

GOV.UK component guide

Examples

Example 1

Default

{{ appBanner({ description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit' } }) }}

Example 2

Banner with a heading

{{ appBanner({ heading: { text: 'Summary' }, description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.' } }) }}

Example 3

Banner with an aside

{{ appBanner({ heading: { text: 'Summary' }, description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.' }, aside: { text: 'Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.' } }) }}

Example 4

With data attributes

{{ appBanner({ heading: { text: 'Summary' }, description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.' }, attributes: { 'aria-label': 'Notice', lang: 'en' } }) }}
