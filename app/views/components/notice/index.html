Components Notice

Displays a notice to convey and highlight information.

GOV.UK component guide

Examples

Example 1

{{ appNotice({ description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit' } }) }}

Example 2

With a heading

{{ appNotice({ heading: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit' } }) }}

Example 3

With a heading and description

{{ appNotice({ heading: { text: 'Summary' }, description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.' } }) }}

Example 4

With multiple paragraphs of description

{{ appNotice({ heading: { text: 'Summary' }, description: { html: '

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

' }, attributes: { 'aria-label': 'Notice', role: 'region' } }) }}

Example 4

With data attributes

{{ appNotice({ heading: { text: 'Summary' }, description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.' }, attributes: { 'aria-label': 'Notice', role: 'region' } }) }}
