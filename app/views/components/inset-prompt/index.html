Components Inset prompt

A prompt to users represented as inset content.

GOV.UK component guide

Examples

Example 1

Default

{{ appInsetPrompt({ title: { text: 'Please meet these requirements before publishing' }, description: { text: 'Document needs a summary before publishing (at least 10 characters)' } }) }}

Example 2

With items passed in as HTML

{%- set descriptionHtml %}
  • Document needs a title before publishing (at least 10 characters)
  • Document needs a summary before publishing (at least 10 characters)
{% endset -%} {{ appInsetPrompt({ title: { text: 'Please meet these requirements before publishing' }, description: { html: descriptionHtml } }) }}

Example 3

Error

{{ appInsetPrompt({ error: true, title: { text: 'There is a problem' }, description: { text: "‘Title here’ was not published" } }) }}

Example 4

List of items array

{{ appInsetPrompt({ title: { text: 'There is a problem' }, items: [{ text: 'Document needs a title before publishing (at least 10 characters)', href: '#' }, { text: 'Document needs a summary before publishing (at least 10 characters)', href: '#' }] }) }}
{% endblock %}
