Alternatives for the ‘parliamentary session’ part of the ‘Is this an official document?’ question

When uploading an attachment we need to know whether the document is ‘official’.

Example 1

Parliamentary session is represented as a dropdown.

{% include "./example-1.html" %}

Example 2

Parliamentary session is represented as a text input. This relies on users understanding the format of a session nummber (YYYY-YY) and they input the correct information. Validation is more complex. We cannot guarantee accuracy of the data, only the syntax.

{% include "./example-2.html" %}

Example 3

House of Commons paper number is prefixed with ‘HC’. Parliamentary session is represented as two text inputs separated by a hyphen.

{% include "./example-3.html" %}

Example 4

Similar to example 3. Parliamentary session is represented as two text inputs separated by the word ‘to’. The user is expected to enter the full year.
